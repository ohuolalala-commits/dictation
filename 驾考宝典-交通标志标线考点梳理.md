# 一、交通道路标线

## 📌 考点总览

交通标线分为 **三大类**：指示标线、禁止标线、警告标线。

## 1.1 实线与虚线（最高频考点）

| 标线 | 颜色 | 作用 | 能否跨越 |
|------|------|------|:--:|
| 白色虚线 | 白 | 分隔**同向**车道 | ✅ 可跨 |
| 白色实线 | 白 | 分隔同向车道 | ❌ 禁止 |
| 黄色虚线 | 黄 | 分隔**对向**车道 | ✅ 可临时跨 |
| 黄色单实线 | 黄 | 分隔对向车道 | ❌ 禁止 |
| 黄色双实线 | 黄 | 分隔对向车道 | ❌ **严格禁止** |
| 黄色虚实线 | 黄 | 分隔对向车道 | 虚侧可越，实侧禁止 |

> 🔑 **口诀**: **白同黄对，实禁虚可跨**

## 1.2 路口标线

| 标线 | 含义 |
|------|------|
| 路口前端**双白实线** | 停车让行线（必须停车瞭望） |
| 路口前端**双白虚线** | 减速让行线（减速观察后通过） |
| 路口内**白色横实线** | 停止线（红灯时停车线） |
| **黄色网状线** | 禁止任何原因停车 |
| **中心圈** | 路口中央圆形标记，引导车辆绕行 |

## 1.3 路缘石标线

| 标线 | 含义 |
|------|------|
| 路缘石上**黄色实线** | 禁止停放车辆 |
| 路缘石上**黄色虚线** | 禁止长时间停车（可临时上下人/装卸货） |

## 1.4 特殊标线

| 标线名称 | 识别特征 | 考点 |
|----------|----------|------|
| **导流线** | V形斜纹线区域 | 引导行驶，**禁止压线或越线** |
| **潮汐车道线** | 两条双黄色虚线 | 可变方向车道 |
| **可变导向车道线** | 锯齿状白色实线框 | 根据信号灯指示选择方向 |
| **车距确认线** | 白色半圆状 / 白色折线 | 确认与前车距离 |
| **纵向减速标线** | 菱形块虚线（∥车道线） | 提醒减速慢行 |
| **横向减速标线** | 垂直于行驶方向的白线 | 提醒减速 |
| **接近障碍物标线** | 黄色斜线填充 | 警告前方有固定障碍物 |
| **立面标记** | 黄黑相间倾斜线条 | 提醒注意高出路面的构造物 |
| **人行横道预告标线** | 白色菱形标线 | 前方接近人行横道 |
| **路面宽度渐变标线** | 黄色渐变 | 车道变窄，注意慢行 |

## 1.5 速度限制标记

| 颜色 | 含义 |
|------|------|
| **黄色**数字 | 最高限速 |
| **白色**数字 | 最低限速 |
| **黑色**数字 | 建议速度 |

## 1.6 道路出入口标线

- **出口标线**: 箭头朝向支路（从主路出去）
- **入口标线**: 箭头朝向主路（从支路进入主路）
- 口诀：**支路进主路 = 入口，主路出支路 = 出口**

# 二、警告标志

## 📌 考点总览

警告标志用于提醒驾驶人注意前方危险路段或特殊情况，形状为 **等边三角形（顶角朝上）**，颜色为 **黄底、黑边、黑图案**。

## 2.1 识别特征

| 特征 | 内容 |
|------|------|
| 形状 | 正等边三角形（顶点朝上） |
| 底色 | 黄色（Y） |
| 边框 | 黑色 |
| 图案 | 黑色 |
| 例外 | 注意信号灯（红黄绿黑四色）、叉形符号（白底红图） |

## 2.2 常见警告标志分类

### 路口/线形类

| 标志 | 考点 |
|------|------|
| 十字交叉 | 前方十字路口，注意来车 |
| T形交叉 | T字路口，注意横向来车 |
| Y形交叉 | Y形分叉路口 |
| 环形交叉 | 前方环岛 |
| 急弯路（左/右） | 前方急转弯，减速慢行 |
| 反向弯路 | 先左后右（或先右后左）两个急弯 |
| 连续弯路 | ≥3个连续急弯 |

### 坡度/路面类

| 标志 | 考点 |
|------|------|
| 上陡坡 / 下陡坡 | 注意坡度，控制车速 |
| 连续下坡 | 长下坡路段，注意刹车 |
| 路面不平 | 前方路面颠簸 |
| 驼峰桥 | 拱度较大，影响视线 |
| 过水路面 | 路面漫水，谨慎通行 |

### 窄路/窄桥类

| 标志 | 考点 |
|------|------|
| 两侧变窄 | 道路两侧同时收窄 |
| 右侧变窄 | 仅右侧收窄 |
| 左侧变窄 | 仅左侧收窄 |
| 窄桥 | 桥面宽度小于路面宽度 |

### 人/车/动物类

| 标志 | 考点 |
|------|------|
| 注意行人 | 前方行人密集区域 |
| 注意儿童 | 学校/幼儿园附近 |
| 注意非机动车 | 自行车/电动车出没 |
| 注意牲畜 | 放牧区，注意牛/羊等 |
| 注意野生动物 | 野生动物保护区域 |
| 注意残疾人 | 残疾人出没区域 |

### 自然/环境类

| 标志 | 考点 |
|------|------|
| 注意落石 | 山体滑坡/落石路段 |
| 注意横风 | 强侧风区域（桥梁/山口） |
| 易滑 | 路面湿滑，易打滑 |
| 傍山险路 | 靠山临崖危险路段 |
| 堤坝路 | 水库/河流堤坝路段 |
| 隧道 | 前方隧道，注意开灯 |
| 渡口 | 前方车辆渡船码头 |
| 村庄 | 途经村庄，注意减速 |

### 铁路道口（重要考点）

| 标志 | 含义 |
|------|------|
| 🚂 火车头图标 | **无人看守**铁路道口 |
| 🚧 栅栏图标 | **有人看守**铁路道口 |
| 一条红线（斜杠） | 距道口 **50米** |
| 两条红线 | 距道口 **100米** |
| 三条红线 | 距道口 **150米** |

> 🔑 口诀：**有栅栏 = 有人看守，火车头 = 无人看守；一杠 50、两杠 100、三杠 150**

### 其他常见警告标志

| 标志 | 考点 |
|------|------|
| 注意信号灯 | 前方有交通信号灯控制 |
| 事故易发路段 | 该路段事故多发，注意安全 |
| 慢行 | 减速慢行 |
| 施工 | 前方道路施工 |
| 注意危险 | 前方存在多种危险因素 |
| 注意潮汐车道 | 可变方向车道 |
| 注意保持车距 | 保持安全跟车距离 |
| 注意合流 | 前方有车辆汇入 |
| 避险车道 | 刹车失灵车辆紧急避险 |
| 注意路面结冰 | 冬季路面可能结冰 |
| 注意雨/雪/雾 | 不利天气条件 |
| 注意积水 | 2022 新增，易积水路段 |
| 建议速度 | 黑底白数字，该路段建议行驶速度 |
| 隧道开车灯 | 进入隧道须开启前照灯 |
| 注意前方车辆排队 | 接近排队路段，提前减速 |

## 2.3 颜色速记

| 颜色组合 | 标志类型 |
|----------|----------|
| 🟡 黄 + 黑 | 警告标志（绝大部分） |
| 🔴🔵🟢⚫ 红/黄/绿/黑 | 注意信号灯（唯一四色标志） |
| ⬜ + 🔴 白 + 红 | 叉形符号（铁路道口） |

# 三、指路标志

## 📌 考点总览

指路标志传递**道路方向、地点、距离**信息。核心区分原则：**有文字/数字的就是指路标志，无文字的是指示标志**。

## 3.1 指路标志 vs 指示标志（必考混淆点）

| 对比项 | 指路标志 | 指示标志 |
|--------|----------|----------|
| 有无文字 | ✅ 有文字/地名/距离 | ❌ 无文字（仅有图形） |
| 一般道路底色 | 蓝色 | 蓝色 |
| 高速/快速路底色 | **绿色** | — |
| 形状 | 长方形/正方形 | 圆形/长方形/正方形 |
| 功能 | 传递方向、地点、距离 | 指示行进规则 |

## 3.2 公路编号颜色

| 颜色 | 含义 | 举例 |
|------|------|------|
| 🔴 红底白字 | **国道** | G107、G318 |
| 🟡 黄底黑字 | **省道** | S101 |
| ⬜ 白底黑字 | **县道** | X005 |
| ⬜ 白底黑字 | **乡道** | Y008 |

> 🔑 口诀：**红国黄省白县乡**

## 3.3 一般道路指路标志

| 类型 | 标志内容 | 考点 |
|------|----------|------|
| 交叉路口预告 | 前方 XXX m → 某地 | 提前告知方向和距离 |
| 交叉路口告知 | 路口处方向指引 | 进入路口前选择车道 |
| 地点距离 | 地名 + 距离（km） | 确认当前位置和距离 |
| 地点方向 | 箭头 + 地名 | 指示前往某地应走的方向 |
| 著名地点 | 风景区、机场等 | 著名地点指引 |
| 分界标志 | "欢迎再来XX" | 行政区域分界线 |
| 街道名称 | 路名牌 | 确认所在道路名称 |

## 3.4 高速公路/城市快速路指路标志

底色为 **绿色**（区别于一般道路的蓝色）。

| 类型 | 说明 |
|------|------|
| 入口预告标志 | 前方进入高速公路的预告 |
| 入口标志 | 高速公路入口处 |
| 出口预告标志 | 依次预告 2.5km、2km、1km、500m、出口 |
| 出口标志 | 出口匝道处 |
| 下一出口预告 | 下一出口名称+距离 |
| 服务区预告 | 服务区名称+距离（P 停车/加油/餐饮等图标） |
| 收费站预告 | 收费站名称+距离 |
| 地点距离标志 | 沿线主要城市名称+距离 |
| 里程牌 | 绿底白字，标明公里数 |
| 百米牌 | 绿底白字，圆形，标明百米数 |
| ETC 车道指引 | 电子不停车收费车道 |

## 3.5 旅游区标志（辅助指路）

| 类型 | 底色 | 含义 |
|------|------|------|
| 旅游指引标志 | 棕色底 + 白色字 | 提供旅游区方向、距离 |
| 旅游符号标志 | 棕色底 + 白色图案 | 表示旅游项目类别（如 🏔️ =登山） |

## 3.6 告示标志（2022版新增/调整）

- 行车安全提醒：如"驾驶时禁用手持电话"
- 道路设施解释：如"前方车道控制"
- 路外设施指引：如医院、学校方向指引

# 四、禁令标志

## 📌 考点总览

禁令标志用于**禁止或限制**车辆、行人的交通行为。绝大部分为 **圆形**，颜色为 **白底 + 红圈 + 红杠 + 黑图案**。

## 4.1 识别特征

| 特征 | 内容 |
|------|------|
| 形状 | **圆形**（绝大多数）；特例：八角形、倒三角形 |
| 底色 | 白色 |
| 边框 | 红色 |
| 杠 | 红色（从左上到右下） |
| 图案 | 黑色（压在红杠之下） |
| 例外 | 解除类标志 — 黑圈 + 黑色图案（无红色） |

## 4.2 核心禁令标志分类

### 道路通行权类

| 标志 | 识别 | 考点 |
|------|------|------|
| 禁止通行 | 红圈内全白（空无一物） | 车辆和行人均禁止通行 |
| 禁止驶入 | 红底 + 白横杠 | 只禁止车辆驶入 |
| 禁止机动车驶入 | 红圈 + 轿车 | 禁止各类机动车 |
| 禁止非机动车进入 | 红圈 + 自行车 | 禁止非机动车 |
| 禁止电动自行车进入 | 红圈 + 电动自行车 | 2022 新增 |
| 禁止行人进入 | 红圈 + 行人 | 禁止行人通行 |

### 方向通行权类

| 标志 | 考点 |
|------|------|
| 禁止向左转弯 | 红圈 + 黑左转箭头 + 红杠 |
| 禁止向右转弯 | 红圈 + 黑右转箭头 + 红杠 |
| 禁止直行 | 红圈 + 黑直行箭头 + 红杠 |
| 禁止向左和向右转弯 | 红圈 + 左右双箭头 + 红杠 |
| 禁止直行和向左转弯 | 组合禁令 |
| 禁止掉头 | 红圈 + 黑掉头箭头 + 红杠 |

### 交通管理措施类

| 标志 | 识别 | 考点 |
|------|------|------|
| 禁止超车 | 红圈 + 两车图案 + 红杠 | 禁止在该路段超车 |
| **解除禁止超车** | 黑圈 + 黑图案（无红色） | 解除之前的超车禁令 |
| 禁止鸣喇叭 | 红圈 + 喇叭 + 红杠 | 禁止按喇叭 |
| 限制速度 | 红圈 + 黑数字 | 最高速度不得超过所示数值 |
| **解除限制速度** | 黑圈 + 黑数字 + 斜线 | 解除之前的限速 |
| **停车检查** | 红圈 + "检"字 | 必须停车接受检查 |
| 禁止危险物品运输车驶入 | 红圈 + 罐车 | 危化品车辆禁行 |

### 停车管理类（高频混淆）

| 标志 | 识别 | 含义 |
|------|------|------|
| 禁止车辆停放 | 红圈 + 蓝底 + ❌（红色交叉） | **禁止**临时或长时间停放 |
| 禁止车辆长时停放 | 红圈 + 蓝底 + ╲（红色单斜杠） | 可**临时停车**，禁止长时 |
| 区域禁止停车 | 长方形 + 区域禁停图案 | 区域内所有道路禁止停车 |
| 区域禁止停车解除 | 长方形 + 禁停解除图案 | 解除区域禁停 |

> ❌ = 禁止停放（一点也不能停）
> ╲ = 禁止长时停放（可临时上下人）

### 通行优先权类

| 标志 | 形状 | 考点 |
|------|------|------|
| **停车让行** | 八角形 | 必须停车瞭望，确认安全后再通过 |
| **减速让行** | 倒等边三角形 | 减速观察，让行后再通过 |
| 会车让行 | 圆形 | 会车时让对方车辆先行 |

> 🔑 八角形 = 停；倒三角 = 让 — 形状是特有的识别特征！

### 道路条件限制类

| 标志 | 含义 |
|------|------|
| 限制宽度 | 超过所示宽度车辆禁止通行 |
| 限制高度 | 超过所示高度车辆禁止通行 |
| 限制质量 | 总质量超过所示数值禁止通行 |
| 限制轴重 | 单轴负荷超过所示数值禁止通行 |

## 4.3 禁令标志关键规律

| 规律 | 说明 |
|------|------|
| **红 = 禁** | 有红色元素 = 禁止/限制 |
| **黑 = 解** | 黑色外圈（无红色）= 解除限制 |
| **杠 = 否定** | 红杠表示"不"，图案被杠压住 = 禁止该行为 |
| **圆 = 禁令** | 绝大多数禁令标志是圆形 |
| **八 = 停** | 八角形是独特的"停车让行" |
| **倒三角 = 让** | 倒三角形是独特的"减速让行" |

# 附录：综合速记口诀

## 四类标志速记

```
红圈红杠 = 禁令标志（禁止、限制）
黄底三角 = 警告标志（提醒、警示）
蓝底无字 = 指示标志（指引行进方式）
蓝底有字 = 指路标志（一般道路，方向地点）
绿底有字 = 指路标志（高速/快速路）
```

## 标线速记

```
白同黄对（白色同向、黄色对向）
实禁虚可跨（实线禁止、虚线可跨越）
火车站无人守，栅栏有人看
一杠五十米，两杠一百米，三杠一百五
导流线禁止压，网状线禁止停
虚侧可越实侧止，双实严格不能碰
```
