# 设计规范

## 主题方向

刺客信条·起源（Assassin's Creed Origins）暗金埃及风

## 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景基色 | `#1a1108` | 深色纸莎草底色 |
| 卡片底色 | `#f7ecd5 → #efe0c0 → #e8d8b4` | 渐变的古纸质感 |
| 头部底色 | `#1a2840 → #0d1b2d → #0a1520` | 深蓝夜空渐变 |
| 金色主色 | `#c8a84e` | 青铜金，用于边框、图标、动画 |
| 金色亮 | `#f4d87c` | 标题文字高光 |
| 金色暗 | `#b8942e` / `#8b6914` | 按钮渐变、进度条 |
| 红色错误 | `#a03a20` | 暗红陶土色，错误/错题 |
| 绿色正确 | `#4a8c5c` | 暗绿青铜色，正确答案 |
| 文字深色 | `#3d2817` / `#2d1f14` | 卡片上可读性高的深棕文字 |

## 字体

- 中文：`Microsoft YaHei` / `PingFang SC`
- 英文输入区：`Georgia` / `Times New Roman`（等宽衬线便于校对）

## 动画

| 动画 | 持续时间 | 缓动 | 用途 |
|------|---------|------|------|
| popIn | 0.5s | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 庆祝弹窗弹出 |
| fadeIn | 0.4s | ease | 反馈面板出现 |
| circleFill | 0.6s | ease | 勾号圆圈填充 |
| checkDraw | 0.4s | ease | 勾号笔画绘制 |
| starFloat | 1.4s | ease-out | 星星 burst 飘散 |
| trophyBounce | 0.8s | ease infinite alternate | 完成奖杯摇摆 |

## 按钮样式规范

- 主要操作按钮：渐变背景 + 底部 2px 深色边框 + hover 上浮 2px
- 次要按钮：透明背景 + 2px 彩色边框 + hover 填色
- 错题本按钮：特殊红色 + 脉冲动画（激活态）
- 圆角：4px（方形古埃及风格，非圆角胶囊）

## 输入框

- 2px 边框 + focus 时变为金色 + 3px 金色外发光 box-shadow
- 文字居中、高度 55px、适合短单词输入

## 词库选择器

- 暗色背景选择框 + 金色文字
- 自定义下拉箭头（内联 SVG）
- 圆角 20px（与其他元素形成风格对比）

## 响应式

- 最大宽度 780px，居中
- 移动端（< 600px）：缩小字号、调整内边距
- iPad 适配：9.7" ~ 12.9" 均在 780px 以内

## 无障碍

- 所有交互元素有可见 focus 样式
- 颜色对比度满足可读性要求
- 键盘操作完整（Tab / Enter / Ctrl+Enter）
