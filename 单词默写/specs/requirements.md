# 项目开发需求

## 项目概述

英语默写本（English Dictation Book）— 单文件 HTML 网页应用，面向中小学生英语默写练习。支持多词库切换、错题复习、键盘音效反馈。

## 核心需求

### 1. 多词库支持
- 用户可从下拉菜单切换词库
- 词库之间独立，各有独立的错题本（localStorage 按 key 隔离）
- 当前已实现词库：
  - 介词短语（高考核心）
  - 北师大版
  - 人教版
  - PEP版·六年级下册
  - 外研社版
  - 小学过去时·不规则动词

### 2. 默写模式
- **默认模式**（大部分词库）：出示中文 → 用户输入英文
- **过去式模式**（pastTense 词库）：出示动词原型 + 中文 → 用户输入过去式
- 支持多正确答案（alt 字段），如 learnt/learned 两种变体都接受

### 3. 错题本
- 答错的词自动加入错题本
- 错题本数据存储在 localStorage，有效期 2 天（过期自动清除）
- 支持错题复习模式：仅显示错题，逐个击破
- 全部纠正后自动退出错题模式

### 4. 辅助功能
- 提示（hint）：显示首字母和长度提示
- 查看答案：直接显示正确答案
- 键盘快捷键：
  - `Enter` — 检查答案并自动跳下一题
  - `Ctrl+Enter` — 仅检查答案（不跳转）

### 5. 反馈系统
- 正确：绿色反馈 + 庆祝动画（彩带、星星 burst）+ 正确音效
- 错误：红底显示正确答案 + 提示信息
- 答对即从错题本移除

### 6. 键盘音效
- 每个按键带机械键盘音效（Web Audio API 合成）
- 不影响浏览器默认音频行为

## 非功能性需求

- 单文件 HTML，无外部依赖，离线可用
- 响应式设计，适配 iPad 和桌面浏览器
- 中文界面，使用微软雅黑字体
- 主题风格：暗金埃及风（Assassin's Creed Origins 主题）
